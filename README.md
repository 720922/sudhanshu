# sudhanshu

A workaround Python script to manipulate object key values in a json file.

## Usage:
```
python3 sudhanshu.py file.json object_name
```

## Supported Actions:
1. Capiatlization
2. Small Case

## Supported Values:
1. Object
2. Object Values
